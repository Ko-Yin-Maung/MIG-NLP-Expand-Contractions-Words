- အတိုကောက် ရေးသားထားသော စကားလုံးများကို ပြန်လည် ချဲ့ထွင် ရေးသားပေးဖို့ အတွက် ရည်ရွယ်ပါတယ်။

- can't -> cannot,
- It's -> It is,
- We'll -> We will,
- won't've -> will not have,
- You'd've -> You would have,
- etc...
